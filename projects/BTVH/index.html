Burger Studio 简介
Burger Studio 于 2020 年创立，起初只是搞一些虚拟机（故中文名为虚拟稽），后来逐渐开始研究其它方面的内容，如 Windows / Linux 系统安装、 Android 刷机等，偶尔会在B站发视频。
在2021年年初，我开始编写 Burger工具站 （原资源站），并不定期地更新。

早期的 Burger Studio 陆续被命名为 Administrator 、 Admin Studios 和 Admin Studio 。
